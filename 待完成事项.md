## 🟡 P2 阶段 — 体验优化（有余力再做）

### 1. PWA 支持（2-3 天）
**做什么**：
- 添加 `manifest.json`（应用名称、图标、主题色）
- 注册 Service Worker，缓存核心资源（HTML/CSS/JS）
- 离线时允许浏览已缓存的文章列表和详情
- 桌面安装提示（"添加到主屏幕"）

**价值**：手机浏览器访问时像原生 App，无网络也能看已读文章

### 2. 批量导入（1-2 天）
**做什么**：
- 上传界面支持选择 ZIP 文件或整个文件夹
- 后端解压后逐个处理，返回批量任务进度
- 前端显示进度条（已完成/总计/失败数）

**价值**：一次性迁移历史资料，不用一个个上传

### 3. Prompt A/B 测试（2-3 天）
**做什么**：
- 同一任务（如"摘要生成"）维护两个 Prompt 版本
- 随机将 50% 请求分配给 A 版本，50% 给 B 版本
- 收集用户 👍👎 反馈，自动统计 A/B 胜率
- 定期自动切换为表现更好的版本

**价值**：让 AI 回答质量持续自动优化

### 4. 用户记忆精细化（2-3 天）
**做什么**：
- 后端 `/api/skill/memory` 真正存储用户画像：
  - `focus_areas`：用户最常问的领域（自动统计）
  - `answer_preferences`：回答长度偏好（简短/详细）、语言风格（专业/通俗）
  - `recent_topics`：最近 20 个话题
- AI 回答时根据记忆调整语气和深度
- 用户可手动编辑记忆（"我更关注货币政策"）

**价值**：AI 越用越懂你的偏好

### 5. 推送通知（1-2 天）
**做什么**：
- 定时扫描 `importance = 'high'` 的文章
- 浏览器 Notification API 推送："今日有 3 篇重要文章"
- 用户可设置推送频率（实时/每日摘要/关闭）

**价值**：不错过重要信息

### 6. 数据源自动发现（1-2 天）
**做什么**：
- 输入网站 URL，自动探测 RSS/Atom feed
- 推荐常见财经源列表（36kr、华尔街见闻、财新等）
- 一键添加推荐源

**价值**：降低添加数据源门槛

## 🔵 阶段四 — 云端 + 多端（第 8-12 周）

### 7. 数据库迁移 SQLite → PostgreSQL（2-3 天）
**做什么**：
- 修改 `.env` 中 `DATABASE_URL` 为 PostgreSQL 连接串
- Alembic 迁移脚本兼容两种数据库
- 数据导出/导入工具

**触发条件**：数据量超过 10GB 或需要多用户并发

### 8. 文件存储迁移 本地 → S3/OSS（2-3 天）
**做什么**：
- 抽象 `StorageBackend` 接口（已预留）
- 实现 `S3StorageBackend` / `OSSStorageBackend`
- 上传文件直传云存储，数据库存 URL

**触发条件**：部署到服务器或团队共享使用

### 9. 向量数据库迁移 ChromaDB → Pinecone（1-2 天）
**做什么**：
- 抽象向量存储接口
- 接入 Pinecone / Milvus 等云向量库

**触发条件**：文章数量超过 10 万篇

### 10. Capacitor 打包 iOS/Android App（1-2 周）
**做什么**：
- 用 Capacitor 将 React 代码打包为原生 App
- 适配移动端手势（下拉刷新、左右滑动切换）
- 推送通知接入 Firebase / 极光推送

**触发条件**：用户强烈需要原生 App 体验

## 🔵 远期（第 12 周+）

### 11. 多用户支持（3-5 天）
**做什么**：
- `users` 表 + JWT 认证
- 每个用户独立的知识库、分类、记忆
- 团队共享模式（管理员可分配权限）

**触发条件**：需要团队共享知识库

### 12. 多领域扩展（3-5 天）
**做什么**：
- `knowledge_domains` 表
- 新增领域 = 填表单（领域名 + Prompt 模板 + 分类树）
- 如：科技分析、法律研究、医疗知识等

**触发条件**：知识库要扩展到财经以外的领域

### 13. 自动化报告（2-3 天）
**做什么**：
- 定时任务：每日/每周自动生成综合分析报告
- 报告内容：热点分布、重要文章汇总、趋势变化
- 支持导出 PDF / Markdown

**触发条件**：用户需要定期汇报或复盘

### 14. 插件市场（1-2 周）
**做什么**：
- `core/crawler.py` 改为插件注册机制
- 开发者可编写自定义爬虫插件（npm 式包管理）
- 内置插件市场页面，一键安装

**触发条件**：数据源类型太多，维护不过来了

## 📋 近期建议优先级

如果你现在只有 1-2 周时间，建议按这个顺序：

| 优先级 | 事项 | 理由 |
|---|---|---|
| 1 | **PWA 支持** | 手机体验提升最明显，工作量小 |
| 2 | **批量导入** | 用户迁移历史资料的刚需 |
| 3 | **用户记忆精细化** | 让"经济大师"真正个性化 |
| 4 | **Prompt A/B 测试** | 长期优化 AI 质量的基础设施 |
| 5 | **推送通知** | 提高用户粘性 |
